# my-first-kotlin-app
# 📱 Mi Tarjeta de Presentación Digital

Esta es mi primera aplicación móvil construida desde cero. El objetivo principal de este proyecto es dominar las bases de la sintaxis del lenguaje Kotlin y comprender los fundamentos de las interfaces de usuario modernas utilizando **Jetpack Compose**.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Kotlin
* **Framework UI:** Jetpack Compose (Google)
* **Entorno de Desarrollo:** Android Studio (Edición nativa para Windows en ARM)

## 🧠 Aprendizajes Clave
* **Sintaxis de Kotlin:** Aprendí a declarar variables mutables e inmutables, y a estructurar funciones básicas.
* **Componentes de Compose:** Utilicé contenedores como `Column` y `Row` para organizar de manera limpia los elementos visuales de la pantalla.
* **Gestión de Estado (`State`):** Implementé lógica básica para que la aplicación reaccione visualmente cuando el usuario interactúa con un botón.
