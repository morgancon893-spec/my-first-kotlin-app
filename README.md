# my-first-kotlin-app
# 📱 My First Android App

This is my very first mobile application built entirely from scratch. The main goal of this project was to master the fundamentals of Kotlin syntax and understand modern user interface principles using **Jetpack Compose**.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Google)
* **IDE:** Android Studio (Native edition for Windows on ARM)

## 🧠 Key Learnings
* **Kotlin Syntax:** Learned how to declare mutable and immutable variables, and structure basic functions.
* **Compose Components:** Used layouts like `Column` and `Row` to arrange UI elements cleanly on the screen.
* **State Management (`State`):** Implemented core logic using `remember` and `mutableStateOf` to make the UI update dynamically when interacting with a button.

*Developed independently as part of my training to apply for Software Engineering roles. Tested and running successfully on a physical Motorola Razr 50.*
